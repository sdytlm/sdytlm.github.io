### Switch 
格式:
``` c
switch (expression) {
    case 常数表达式: statements
    case 常数表达式: statements
    default: statements
}
```
* default 不是必须有的.
* case 和default 的顺序可以是任意的.
* falling through 执行顺序: 一个case 执行完， 还会去test另外的case 并执行. 除非有break 退出.
``` c
switch(c){
case '0': case '1': case '2:
    ndigit[c-'0']++;
    break;
case ' ':
case '\n': 
    nwhite++;
    break;
default:
    nother++;
    break;
```
